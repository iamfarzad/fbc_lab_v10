# Stress Test Prompts for SSE Streaming

## Prompt 1: Long Detailed Response (Tests Multiple Chunks)
```
I'm a small business owner with 25 employees in the manufacturing industry. We're currently spending about $50K annually on various software tools that don't integrate well. I'm interested in AI automation but I'm concerned about:
1. The upfront cost and ROI timeline
2. Whether my team has the technical skills needed
3. How long implementation takes
4. What happens if it doesn't work out
5. Security and data privacy concerns

Can you provide a comprehensive, detailed analysis addressing each of these concerns with specific examples, case studies, and actionable recommendations? Please be thorough and include specific numbers, timelines, and real-world scenarios.
```

## Prompt 2: Tool Call Trigger (Tests Metadata Events)
```
I need you to research the latest AI automation trends for small businesses in 2024. Look up current market data, pricing benchmarks, and success rates. Then create a detailed proposal for a manufacturing company with 25 employees and a $50K annual software budget.
```

## Prompt 3: Complex Multi-Step Reasoning (Tests Reasoning Events)
```
I'm evaluating whether to invest in AI automation. Help me think through this decision step by step:
- First, analyze my current situation: 25 employees, manufacturing, $50K software budget, limited technical expertise
- Then, identify the top 3 automation opportunities that would have the biggest impact
- Calculate the potential ROI for each, including payback period
- Assess the risks and mitigation strategies
- Finally, create a phased implementation plan with milestones

Show your reasoning process for each step.
```

## Prompt 4: Maximum Stress Test (All Features)
```
I'm the CEO of a 25-person manufacturing company considering AI automation. I need a comprehensive strategic analysis that includes:

1. Current state assessment using my company profile (25 employees, manufacturing, $50K software budget)
2. Market research on AI automation trends and pricing for companies my size
3. Detailed ROI analysis with 3 scenarios: conservative, realistic, optimistic
4. Risk assessment with mitigation strategies
5. Implementation roadmap with phases, timelines, and milestones
6. Vendor comparison and recommendations
7. Change management plan for my team
8. Success metrics and KPIs to track

Please be extremely detailed, include specific numbers, case studies, and show your reasoning at each step. This is a critical business decision.
```
